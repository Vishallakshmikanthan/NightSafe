# NightSafe Sample Data

This folder contains sample CSV datasets used for development and ML training.

## Files

| File | Description |
|---|---|
| `crime_data.csv` | Simulated crime incident records (NYC sample) |
| `lighting_data.csv` | Street lighting coverage data |
| `route_segments.csv` | Pre-computed route segments with features |
| `chennai_street_data.csv` | Chennai hourly street safety data (55 streets × 5 hours) |
| `simulate_chennai.py` | Generator script for Chennai street data |

## Chennai Street Data

Generated by `simulate_chennai.py`. 275 rows covering 55 real Chennai streets
across the 8 PM – midnight window.

**Columns**: `street_id`, `street_name`, `hour`, `footfall`, `lighting_status`,
`liquor_distance`, `crime_score`

**Realistic patterns baked in**:
- Footfall drops from ~200 avg at 8 PM → ~28 avg at midnight
- Liquor proximity spikes at 10 PM (TASMAC closing time) — avg distance halves
- Crime score rises through the night, amplified when street lights fail
- Per-street deterministic profiles ensure reproducibility (`--seed`)

Regenerate: `python data/simulate_chennai.py`

## Data Sources (Production)

In production, these datasets would be sourced from:
- **Crime data**: Chennai Police / NCRB open data, NYC Open Data
- **Lighting data**: Municipal street light inventories (GCC)
- **Liquor outlets**: TASMAC store registry + OpenStreetMap
- **Footfall**: Google Popular Times, municipal footfall counters
- **Route segments**: OpenStreetMap + Google Maps API
